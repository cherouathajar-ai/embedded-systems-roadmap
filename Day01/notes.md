## Day 1 notes

### What is source code?
its simply the set of instructions that a developper writes to run a software, so its simply the line of codes
### What does a compiler do?
its like a translator that translate human code into low level machine code 
### What is an executable?
An executable is a computer file that contains encoded instructions. When opened or triggered, the operating system directly executes these instructions to run a program or app.
## Variable
a human-readable name assigned to a specific memory location in the computer's RAM where a value can be stored, retrieved, and manipulated during program execution
## int
 a data type which stores whole numbers. Most implementations will give the int type 32 (4 bytes) bits, but some only give it 16 bits (2 bytes)
## double
a data type used to store high-precision floating-point data or numbers (up to 15 to 17 digits). It is used to store large values of decimal numbers.
## Integer division
Integer division means dividing two whole numbers (int) in C.
When both numbers are integers, C removes the decimal part
## printf
is a function used to print out formatted output
## scanf
function used to read formatted input from the standard input device ( stdin )
## Format specifiers
%d means: for int
%f means: for float
## main()
the designated entry point for any hosted C program, serving as the exact location where the operating system begins executing your code.
