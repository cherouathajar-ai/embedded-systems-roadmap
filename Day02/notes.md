# Day 2 — Functions, Loops and Ohm's Law

## Function

A function is a reusable block of code that performs one specific task.

## Parameter

A parameter is a variable written in a function definition that receives a value.

## Argument

An argument is the actual value or variable passed when calling a function.

## Return value

A return value is the result sent back by a function.

## Function prototype

A prototype tells the compiler the function name, return type, and parameters before the function is used.

## switch statement

A switch selects one block of code depending on the value of a variable.

## break

break stops the current case and prevents execution from continuing into the next case.

## do-while loop

A do-while loop executes its body at least once and repeats while its condition is true.

## scanf return value

scanf returns the number of values successfully read. If one value is expected, a return value different from 1 means the input failed.

## Division by zero

Division by zero is undefined, so resistance and current must be checked before using them as divisors.

## Ohm's Law

- V = I × R
- I = V / R
- R = V / I

## Electrical Power

- P = V × I
- P = I² × R
- P = V² / R
