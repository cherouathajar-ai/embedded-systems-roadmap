# Day 3 — Arrays, Sensor Data and Voltage Dividers

## Array
is a data structure that stores a fixed-size, sequential collection of elements of the same type.
## Array index
Array indices start at zero in C, and go to one less than the size of the array. For example, a five element array will have indices zero through four. This is because the index in C is actually an offset from the beginning of the array.
## for loop
a control flow statement that is used to repeatedly execute a block of code as many times as instructed.
## Array length
To find the length of a statically allocated array in C, use the formula sizeof(array) / sizeof(array[0]).

## Voltage divider
A voltage divider is a simple, fundamental electronic circuit that steps down a higher input voltage to a lower output voltage. It typically consists of two resistors connected in series, with the output voltage measured across the second resistor.
