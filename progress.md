## Day 2 Reflection

### What I learned

I learned how to divide a program into reusable functions and how to use function prototypes. I also learned how switch and do-while can create a repeating menu.

### Problems I encountered

I initially placed function definitions inside main and placed the do-while closing condition before all switch cases.

### How I solved them

I reorganized the program into prototypes, main, and function definitions. I also corrected the order of the switch and loop braces.
